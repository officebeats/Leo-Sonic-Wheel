# 🎡 Shadow's Hamster Wheel

An endless runner featuring Shadow the Hedgehog inside a hamster wheel, inspired by **Canabalt** and **Donkey Kong Country's mine cart levels**.

## 🎮 Play Now
**[Play on GitHub Pages](https://officebeats.github.io/Leo-Sonic-Wheel/)**

## Controls
- **Desktop:** `Space` / `Up Arrow` / `W` to jump
- **Mobile/Tablet:** Tap anywhere to jump

## Features
- 🎡 DKC mine cart mechanic — wheel + Shadow jump as one unit
- 🏙️ Procedural rooftop generation with parallax city skyline
- 💍 Collectible rings with Sonic-style chime sound
- 📈 Progressive difficulty — gentle start, gradual ramp
- 📱 Mobile & tablet touch support
- ✨ Screen shake, dust particles, speed lines

## Status
🚧 **Alpha** — early playable build

## Tech
Pure HTML5 Canvas + JavaScript. No frameworks, no dependencies.
